## 快速开始

### 获取API密钥

1. 登录 [bigmodel 平台](https://open.bigmodel.cn/usercenter/)
2. 进入「[API密钥](https://open.bigmodel.cn/usercenter/proj-mgmt/apikeys)」管理页面
3. 创建或复制您的API密钥
